About Us

Remember everything important, stay organized, wherever you are

NOTAS is a free web service, developed for institutes, that aims to simplify creating and distributing assignments in a paperless way The primary purpose of NOTAS is to streamline the process of sharing files between teachers and students.NOTAS ties various other web services to help educational institutions go to a paperless system.

Images, pdfs and many more to be implemented can be uploaded, shared and downloaded to local storage. Functions such as like, comment, delete (Only personal posts) and share functionality is present in NOTAS. We have used best search algorithm to search all over database within a seconds.
